# com

Just another website.

## Docker Usage

Obviously needs `docker`.

### Building

- `docker build -t dodekeract/com .`

### Running

- `docker run -p 8080:8080 -d dodekeract/com`

## Development Usage

Needs `node` and `npm`.

### Installing

- `npm run install`

### Watching

- `npm run watch`

### Building

- `npm run build`

### Starting

- `npm run start`
